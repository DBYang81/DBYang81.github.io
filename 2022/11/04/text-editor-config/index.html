OhMyZshell

On Ubuntu, it install new zsh in order to make things look and behave in a suitable way.
ohmyzsh basic installation can be run one of following commands in the terminal. Installation can be via the command-line with either curl wget.

  • curl sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
  • wget sh -c "$(wget -O- https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
  • fetch sh -c "$(fetch -o - https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

Simply, on my Ubuntu, I install it by curl but first of all it need to install curl sudo apt install curl.

vim ~/.zshrc config the theme by modify the ZSH_THEME="robbyrussell" and source ~/.zshrc to display the theme. we can also change the ZSH_THEME="random" to select various theme in terminal.

GDB as Debugger

GDB, the GNU Project Debugger, allows to see what is going on inside another program while it executes or what another program was doing at the moment it crashed.

GDB can do four main things to help catch the bugs in the act:

  • Start program, specifying anything that might affeect its behavior
  • Make your program stop on specified conditions
  • Examine what has happened, when program has stopped.
  • Change things in the program, so you can experiment with correcting the effects of one bug and go on to learn about another.
    GDB can run on most popular UNIX and Microsoft Windows variants, as well as on Mac OS X.

GDB Supported Language

GDB supports languages as follows:

  • Ada
  • Assembly
  • C / C++
  • D
  • Fortran
  • Go
  • Objective-C
  • OpenCl
  • Modula-2
  • Pascal
  • Rust

GDB Installed in Ubuntu

GDB install - sudo apt install gdb
